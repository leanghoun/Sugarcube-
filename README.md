# Sugarcube+
Based on [Ruiqimao's SugarCube](https://github.com/ruiqimao/SugarCube). I have extended the build dimensions to 184mm x 184mm x 180mm. The frame, gantry, and panels have also been updated accordingly. Unfortunately, directly stretching the CAD to maintain the same pleasing aspect ratios has resulted in custom linear rails, bed, and silicon heater. Combined with the extensive laser-cut aluminum and polycarbonate panels, this adds up to quite an expensive build.
